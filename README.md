# Calculator 프로젝트

## 프로젝트 소개
제네릭과 Enum을 활용한 간단한 사칙연산 계산기

## 주요 기능
- 사칙연산 (덧셈, 뺄셈, 곱셈, 나눗셈)
- 계산 결과 저장 및 조회
- 특정 값보다 큰 결과 필터링
- 가장 오래된 결과 삭제

## 파일 구조
- `Calculator.java` - 계산 로직 및 결과 관리 클래스
- `OperatorType.java` - 연산자 타입을 관리하는 Enum
- `App.java` - 사용자 입력을 받아 계산을 실행하는 메인 클래스

## 주요 특징
- 제네릭 사용: `<T extends Number>`로 다양한 숫자 타입 지원
- Enum 활용: 연산자를 안전하게 관리
- Stream API: 결과 필터링에 함수형 프로그래밍 적용
- 예외 처리: 0으로 나누기, 잘못된 연산자 입력 처리

## 실행 방법
1. `App.java` 실행
2. 숫자와 연산자 입력
3. 결과 확인 후 계속 진행 또는 종료

## 사용 예시
```
첫 번째 숫자를 입력하세요: 10
사칙연산 기호를 입력하세요: +
두 번째 숫자를 입력하세요: 5
결과: 15.0
```
